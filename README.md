# U-Net Climate Image Reconstruction — IPSL-AID Mini Project

This repository contains a small notebook-based mini-project exploring the use of a U-Net model for climate image reconstruction.

The objective is to test the potential of U-Net architectures for reconstructing climate-related images and fields. For this work, I used the original **IPSL-AID** project as a reference and starting point.

Original project: [IPSL-AID](https://github.com/kardaneh/IPSL-AID)  
Paper: [IPSL-AID: Generative Diffusion Models for Climate Downscaling from Global to Regional Scales](https://arxiv.org/abs/2604.03275)

## Notebook

The notebook is available in this repository:

- `NODIER_David_project_ERA5.ipynb`
